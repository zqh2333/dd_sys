# 🚀 终极交互式跨平台 DD 重装脚本

这是一个将**“极其硬核的底层能力”**与**“极其友好的交互体验”**完美融合的全自动系统重装脚本。

不仅支持 19 种 Linux 系统和 Windows 官方原版镜像的任意方向互刷，还全程提供友好的中英文交互引导，无需记忆任何繁杂的命令行参数，小白也能“一路回车”完成极其复杂的服务器重置。

---

## ✨ 核心特性 (Features)

### 🖥️ 极致的用户交互体验
- **✅ 交互式配置**：通过友好的终端问答，轻松设置 **主机名、密码、端口、时区、Swap 大小**。
- **✅ 一键无脑回车**：支持全程按 `Enter` 使用预设默认值，告别长串命令。
- **✅ 自动识别网络**：运行伊始自动抓取并展示当前的公网 IP、网关与子网掩码，做到心中有数。
- **✅ 自动 BBR 加速**：Linux 系统下支持一键开启 BBR 网络加速，提升网络吞吐。

### 🪓 硬核的底层技术引擎
- **💻 跨平台任意互刷**：完美支持 `Linux to Linux`、`Linux to Windows`、`Windows to Linux`。
- **🪟 Windows 官方原版**：**坚决不用自制镜像包！** 脚本自动拉取微软官方原版 ISO，并在安装时**全自动脱机注入 VirtIO 等公有云底层驱动**，开机即连网。
- **🌐 变态级网络自适应**：智能设置静态/动态 IP，完美支持 `/32`, `/128` 掩码、网关不在子网范围内、纯 IPv6 机器等极端网络。
- **💽 分区表 ID 绝对防误删**：全程使用 **分区表 ID (Partition Table ID)** 识别目标硬盘，彻底杜绝在多硬盘环境下写错硬盘的灾难。
- **🧠 极致低配适配**：专为小鸡优化，运行所需内存甚至比各官方的 `netboot` 还要低。
- **🛠 架构全覆盖**：无缝兼容传统 **BIOS** 和现代 **EFI** 引导，完美支持 **ARM (aarch64)** 架构。
