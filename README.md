## ✨ 核心特性 (Features)

### 🖥️ 极致的用户交互体验 (二合一)
本项目包含两套独立流程，可分阶段完美闭环：
- **阶段一 [一键重装]**：全程引导式选择系统、配置密码和端口，确认后自动进入无人值守格式化安装。
- **阶段二 [一键精装]**：重装完成后再次运行脚本，一键修改 **主机名、系统时区、Swap 大小**，并可一键开启 **BBR 网络加速**。
- **精美可视化**：全脚本引入 ANSI 终端色彩高亮（Info/Success/Warn/Error），状态清晰，赏心悦目。

### 🪓 硬核的底层技术引擎
- **💻 跨平台任意互刷**：完美支持 `Linux to Linux`、`Linux to Windows`、`Windows to Linux`。支持 19 种 Linux 发行版。
- **🪟 Windows 官方原版直装**：**坚决不用带后门的自制镜像包！** 自动拉取微软官方原版 ISO，并在安装时**全自动脱机注入 VirtIO 等公有云底层驱动**，开机即连网。
- **🌐 变态级网络自适应**：智能探测并设置静态/动态 IP，完美支持 `/32`, `/128` 掩码、网关不在子网范围内、纯 IPv6 机器等极端网络。
- **💽 分区表 ID 绝对防误删**：全程使用 **分区表 ID (Partition Table ID)** 识别目标硬盘，彻底杜绝在多硬盘环境下写错硬盘的灾难。
- **🧠 私有化稳定控制**：底层引擎已 Fork 至本仓库下 (`zqh2333/reinstall`)，彻底切断上游不可控更新风险，保障 100% 稳定运行。
